# Interlingua Resources
A list of resources to learn Interlingua, a constructed language that connects European languages.

## Online Tools
[Verbix (conjugate verbs)](https://docs.verbix.com/Languages/Interlingua)

[Parlator de Interlingua (text to speech)](http://www.parlator.org/)

[Tatoeba (sentences in Interlingua)](https://tatoeba.org/en/sentences/show_all_in/ina/none)

[Forvo (Interlingua pronunciation dictionary)](https://forvo.com/languages/ia/)


### Translators
[Online Interlingua-English Translator](https://interlingua-translator.vercel.app/)

[Stars21 Interlingua Translator](https://www.stars21.com/translator/interlingua/)

[Online English Interlingua Translator](https://www.interlinguatranslate.org/dashboard/docs/pokus5.html)
